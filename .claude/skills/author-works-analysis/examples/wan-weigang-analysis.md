## 思想演变总结

### 三大阶段

#### 阶段1：理工科思维奠基（2016-2017）
**关键词：** 建立框架
- 从直觉思维转向数据思维
- 建立STEM方法论
- 开创知识服务模式
- **代表作：** 《万万没想到》《智识分子》《精英日课》1-2季

#### 阶段2：不确定性应对（2018-2019）
**关键词：** 深化拓展
- 引入博弈论和概率论
- 跨学科思维整合
- 从确定性到不确定性
- **代表作：** 《你有你的计划》《精英日课》3-4季

#### 阶段3：专业领域深化（2020-2022）
**关键词：** 系统化大成
- 物理学前沿（相对论）
- 博弈论实际应用
- 科学思维方法论
- **代表作：** 《相对论究竟是什么》《博弈论究竟是什么》《科学思考者》

### 核心概念演进

```
2016: STEM Thinking →
2018: Game Theory →
2022: Scientific Thinking（方法论大成）
```

### 关注领域转移

```
2016-2017: 个人成长 + 认知偏差
    ↓
2018-2019: 决策方法 + 跨学科
    ↓
2020-2022: 科学前沿 + 方法论
```
