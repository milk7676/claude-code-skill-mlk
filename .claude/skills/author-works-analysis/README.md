## 🎯 简介

这是一个用于**系统性分析作者作品**的工具和技能包（Skill），帮助你：

- 📖 **全面梳理**作者的作品列表
- 🧠 **深度分析**核心观点和思想演变
- 📊 **创建表格**系统化展示作品信息
- 💡 **提供指导**个性化阅读建议
- 🔍 **发现脉络**追踪思想发展历程

---

## ✨ 核心功能

### 1. 系统化作品分析
- 按时间排序所有作品
- 提取核心主题和观点
- 识别关键概念和案例
- 建立作品关系网络

### 2. 思想演变追踪
- 识别发展阶段
- 追踪思维模式变化
- 发现内在联系
- 总结演进规律

### 3. 飞书表格创建
- 23个标准字段设计
- 自动化数据填充
- 可视化展示
- 便于筛选和排序

### 4. 阅读指导系统
- 必读作品推荐
- 入门/进阶/高级分类
- 个性化阅读路径
- 实用性评估

## 📖 使用方法

### 基础用法

在 Claude Code 中直接调用：

```
用户：帮我整理万维钢的作品

系统自动执行：
1. 信息收集：11部作品（2016-2022）
2. 表格创建：23个字段
3. 数据填充：完整信息
4. 分析总结：思想脉络 + 必读推荐
```

### 高级用法

```
用户：分析尤瓦尔·赫拉利《人类简史》三部曲的思想演变

系统执行：
1. 对比三部作品核心观点
2. 识别科幻概念演进
3. 总结思想发展脉络
4. 提供阅读顺序建议
```

## 📊 标准字段设计

### 六大类别（23个字段）

#### 1. 基本信息
- 序号、作品名称、出版年份、作品类型、字数篇幅

#### 2. 内容分析
- 核心主题、主要观点、关键概念、代表案例

#### 3. 思维演变
- 思维方式、关注领域、理论深度、实用程度

#### 4. 时代背景
- 时代背景、目标读者、社会影响

#### 5. 思想脉络
- 与前作关系、思想演进、新增内容

#### 6. 评价体系
- 个人评分、推荐指数、阅读建议、备注

详细说明：[reference/field-design.md](reference/field-design.md)

## 🎓 完整案例

### 万维钢作品分析

**作者：** 万维钢（同人于野）
**作品数量：** 11部（2016-2022）
**必读推荐：** 3部

#### 思想演变三阶段

**第一阶段：理工科思维奠基（2016-2017）**
- 《万万没想到》- 建立STEM思维框架
- 《智识分子》- 引入系统思维
- 《精英日课》1-2季 - 知识服务模式

**第二阶段：不确定性应对（2018-2019）**
- 《你有你的计划，世界另有计划》- 博弈论和概率思维
- 《精英日课》3-4季 - 跨学科整合

**第三阶段：专业领域深化（2020-2022）**
- 《相对论究竟是什么》- 物理学前沿
- 《博弈论究竟是什么》- 实际应用
- 《科学思考者》- 方法论大成

#### 三大必读
1. **《精英日课第一季》**（9.5/10）- 入门首选
2. **《你有你的计划，世界另有计划》**（9.5/10）- 决策思维
3. **《科学思考者》**（9.5/10）- 方法论大成
