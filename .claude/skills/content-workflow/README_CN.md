# Content Workflow - 内容创作工作流 Skill

## 📖 简介

这是一个完整的内容创作工作流 skill，从需求收集到最终交付演示文稿的全流程自动化。适用于需要同时创建文章、PPT 和演讲逐字稿的场景。

## 🎯 核心功能

1. **交互式需求收集** - 自动询问主题、受众、长度、风格等关键信息
2. **资料处理** - 处理用户上传的参考文档（PDF、Word、Markdown 等）
3. **文章大纲生成** - 创建结构化的文章大纲
4. **文章写作** - 基于大纲和参考资料撰写完整文章
5. **PPT 大纲生成** - 将文章转换为演示文稿大纲
6. **演讲逐字稿** - 为每页 PPT 生成详细的演讲脚本
7. **PPT 生成** - 自动调用 pptx skill 生成最终演示文稿

## 🚀 使用方法

### 自动触发

直接描述你的需求，skill 会自动识别并启动：

```
"帮我创建一个关于人工智能发展的演示文稿"
"我需要做一个产品发布演讲，包含文章和PPT"

/content-workflow
```

## 📋 工作流程详解

### 第 1 步：需求收集

Skill 会询问以下信息：

- **主题** - 内容的主题是什么？
- **目标受众** - 谁会阅读/观看？
- **内容类型** - 教育、说服、信息、激励？
- **长度要求** - 文章字数、PPT 页数
- **关键点** - 必须覆盖的核心信息
- **风格** - 正式、随意、专业、学术？
- **参考资料** - 是否有文档需要上传？

### 第 2 步：资料上传

上传你的参考文档：
- PDF 文档
- Word 文档
- Markdown 文件
- 研究论文、文章、笔记

Skill 会自动提取关键信息、事实、数据点和引用。

### 第 3 步：文章大纲生成

创建结构化的文章大纲，包括：
- 吸引人的标题建议
- 引言（钩子、背景、论点）
- 3-5 个主要章节
- 结论和行动号召

### 第 4 步：文章写作

基于大纲撰写完整文章：
- 引人入胜的开头
- 完整的主体段落
- 流畅的过渡
- 强有力的结论

### 第 5 步：PPT 大纲生成

将文章转换为演示文稿大纲：
- 每页一个核心观点
- 每页 3-5 个要点
- 视觉元素建议
- 逻辑流程设计

### 第 6 步：演讲逐字稿

为每页 PPT 创建详细的演讲脚本：
- 口语化表达
- 故事和例子
- 过渡语
- 强调点标注
- 每页约 150-200 字

### 第 7 步：PPT 生成

调用 pptx skill 自动生成：
- 专业的设计布局
- 视觉层次
- 演讲者注释集成

## ⚙️ 配置选项

### 文章长度
- 短篇：500-800 字
- 中篇：1000-1500 字
- 长篇：2000+ 字

### 演示文稿长度
- 快速：5-10 分钟（5-10 页）
- 标准：15-20 分钟（15-20 页）
- 延长：30+ 分钟（25+ 页）

### 风格选项
- **正式/学术** - 专业术语，客观语言，引用规范
- **专业/商务** - 简洁明了，数据驱动，行动导向
- **对话/教育** - 通俗易懂，故事丰富，互动性强
- **激励/启发** - 情感共鸣，修辞手法，愿景导向

## ❓ 常见问题

**Q: 可以修改已经生成的内容吗？**
A: 可以！每个步骤完成后都可以要求调整，然后继续下一步。

**Q: 支持哪些文档格式？**
A: PDF、Word (.docx)、Markdown (.md)、纯文本等。

**Q: 生成的 PPT 可以自定义样式吗？**
A: 可以！在第 7 步生成 PPT 时，可以指定颜色主题、布局风格等。

**Q: 如何提高输出质量？**
A: 提供详细的参考资料和明确的需求说明，质量会更高。

**Q: 可以生成多种语言吗？**
A: 可以，在需求收集时指定目标语言即可。
