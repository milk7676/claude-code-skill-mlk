# Claude Code Skill - MLK

这是一个 Claude Code 自定义技能项目。

## 项目说明

此项目用于创建和管理 Claude Code 的自定义技能。

## 安装使用

1. 将此项目克隆到本地的 `~/.claude/skills/` 或项目的 `.claude/skills/` 目录
2. 重启 Claude Code 会话
3. 技能将自动加载并可用

## 技能结构

```
.claude/skills/
├── SKILL.md          # 技能定义文件
├── skill.js/ts/py    # 技能实现代码
└── README.md         # 说明文档
```
